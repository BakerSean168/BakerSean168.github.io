# 颜色结构

- 主色（1-2种）：用于品牌标识、核心按钮或标题，占比约60%。
- 辅助色（2-3种）：用于次级元素、图标或背景，占比约30%。
- 强调色（1种）：用于高亮关键信息（如CTA按钮、链接），占比约10% 
  例如，Adobe Color的互补色方案中，主色与互补色形成强烈对比，适合吸引注意力 

# UI 设计

遵循60-30-10法则：主色覆盖大面积背景或导航栏，辅助色用于卡片、边框，强调色用于交互元素  

# 注意

- 避免高饱和度
- 注意对比度与可读性
